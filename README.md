# OpportuniHub

## Descripción
OpportuniHub es una aplicación web donde las personas podran ver diferentes oportunidades disponibles, como trabajos, internados, becas y proyectos de investigación. En este las personas podrán suscribirse con su email para recibir las últimas oportunidades que fueron añadidas. Por otro lado, estos podrán crearse una cuenta para convertirse en contribuidor y poder publicar también oportunidades. En el caso de los administradores tendrán el control completo de todas las oportunidades y cuentas disponibles en la aplicación web.

## Para guardarlo
1. Al descargar la carpeta zip, busca la carpeta xampp en tu ordenador y luego busca otra carpeta llamada htdocs.
2. Crea una carpeta llamada ccom4019 o la que guste y dentro de esa carpeta coloca la que se extrajo de la carpeta zip del proyecto.

## Crear base de datos
1. Activar el Apache y MYSQL en el Xampp.
2. En tu navegador web, entra a (https://localhost/phpmyadmin/)
3. Luego iras a la pestaña que dice Importar y en la carpeta model que esta en el proyecto, escogera el archivo que se llama opportuniHubDB.sql, luego de eso presionas el boton de Importar.

## Como utilizarla
1. Deberas entrar al navegador web y entrar en (https://localhost/ccom4019/project)
2. IMPORTANTE: el url puede cambiar si el nombre de las carpetas no son como se presentan.
